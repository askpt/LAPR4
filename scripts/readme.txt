for the script build_tests, you must create the folder tmp-build-tests in the source path project

for the script makejar, you must create the folders tmp-build and dist in the source path project

in order to run the tests, you must add the path of the file the run_tests. For example, if your test file is in the path src-tests/csheets/ext/share and the name of java file is ConnectionTest, you must add in the run_tests the following argument: csheets.ext.share.ConnectionTest
